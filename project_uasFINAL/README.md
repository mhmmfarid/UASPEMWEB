EcoCraft Store

EcoCraft Store adalah sebuah project website toko online sederhana yang dibuat sebagai bagian dari tugas/materi pemrograman web. Website ini bertemakan penjualan produk-produk ramah lingkungan yang terbuat dari barang daur ulang.

Demo live: https://uaspemweb-seven.vercel.app/
EcoCraft Store adalah website toko online statis (front-end) yang menampilkan:

Halaman Home, About, dan Kontak
Daftar produk yang dapat dipilih berdasarkan kategori seperti Tas, Selimut, Tempat Pencil, dll
Keranjang belanja sederhana
Desain yang ramah pengguna dan fokus pada produk daur ulang
Halaman footer dengan informasi hak cipta
Website ini cocok untuk tugas pemrograman web dasar atau untuk belajar membuat UI toko online tanpa backend.
Fitur
✔️ Tampilan produk berdasarkan kategori
✔️ Keranjang belanja sederhana
✔️ Desain responsif minimal
✔️ Tayang secara online melalui Vercel
